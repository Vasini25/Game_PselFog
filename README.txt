*PARA BAIXAR*

Baixe todos os arquivos e execute 'PselFog.exe', contido na pasta 'Build'


*COMO JOGAR*

"A" -> segurar para andar para a esquerda
"D" -> segurar para andar para a direita
"Shift" -> segurar para correr enquanto segura uma tecla para andar
"Space" -> toque para pular/segure para pular mais alto
"Esc" -> toque para retornar ao menu inicial

Caveiras são suas inimigas, tocar nelas é letal, ao menos que seja no crânio!

Plataformas rosa são trampolins e o fazem pular mais alto ao pisar nelas, 
a menos que tente pular ao mesmo tempo.

Pule em direção a uma parede para se agarrar a ela e ter impulso para o 
lado contrário a ela quando pular.

Objetivo: chegar ao final da fase.
